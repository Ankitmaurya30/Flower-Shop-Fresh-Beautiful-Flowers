🌸 Flower Shop | Fresh & Beautiful Flowers

A modern and responsive Flower Shop Website built to showcase beautiful flower collections, bouquets, and gifting products with an elegant user interface. This project focuses on clean design, smooth user experience, and responsive layouts for all devices.

🚀 Features:
<br>
🌷 Beautiful landing page design
<br>
🛍️ Product showcase section
<br>
🔍 Search functionality
<br>
🛒 Add to Cart feature
<br>
📱 Fully responsive design
<br>
💌 Contact section

🛠️ Tech Stack:
<br>
Frontend: HTML5, CSS3, JavaScript
<br>
Styling: CSS / Tailwind CSS
<br>
Version Control: Git & GitHub
